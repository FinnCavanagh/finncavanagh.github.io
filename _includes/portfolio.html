WDI Project Three
Plannerr

Plannerr

This is a distraction-free group planning app. Create events and invite friends via Facebook, the app allows each friend to suggest a different activity via the Google Maps API.

Mobile-first, the app was built using Node.js, Express and MongoDB. We also used Ajax and Underscore. A win was successfully integrating the Google Maps API and a seamless Facebook login system.

WDI Project Four
Power

Power App

In the noisy world of messaging apps it's rare to actually say something that has any power. Pulling data from a quotes API I designed a messaging app that that sends meaningful messages using quotes as inspiration. Edit as you wish and then send as SMS.

Integrated with Twilio, with authentication using Passport & JWT the MEANstack app has a clean design using Skeleton, jQuery and Angular.

In this Hackathon we had a lot of fun coming up with a friendly festive game for all the family. A group has 60 seconds to simultaneously draw the Christmas item. Hilarious and strangely addictive.
